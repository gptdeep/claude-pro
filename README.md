1. Caude 到底是个啥？
它是由美国人工智能公司 Anthropic 开发的新一代大语言模型，名字致敬了信息论先驱克劳德·香农。它目前分化出了不同的模型版本体系，比如平衡性能的 Sonnet、主打速度的 Haiku，以及能力更强的 Opus。相比传统聊天机器人，它在自然语言理解、长文本处理、编程能力以及复杂推理上表现更为突出。
2. 它背后是哪家公司？
是 Anthropic 公司，2021 年在美国成立的。这家公司的核心团队里有不少以前参与过大型语言模型研发的成员，他们做 Claude 的核心发展方向就是安全、可控和可靠的人工智能。
